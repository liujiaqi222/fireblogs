# 项目介绍

使用vue 3 + vuex + vue-router + scss，用vite 2.0打包 ，并用firebase当做数据库，构造一个博客的响应式前端页面。

> firebase 为谷歌提供的服务，需要代理才能正常使用



目前：已经实现注册登录，新增，预览，修改，删除博客，添加管理员等等功能。
